# Plain Java Project

Contoh proyek Java sederhana (Plain JDK).

Cara compile & run (PowerShell):

1. Compile:
   `javac -d out src/Main.java`

2. Run:
   `java -cp out Main`

Atau jalankan script:

```
./scripts/compile_and_run.ps1
```

Di VS Code: buka folder proyek, klik tombol Run di atas `main` method atau gunakan panel Run.
